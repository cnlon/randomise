# randomise

To generate random numbers.

## Installation

```bash
npm install -g randomise
```

## Usage

```bash
randomise
# 92
```

or

```bash
randomise count=10 min=10 max=100 precision=1 separator=', ' # randomise c=10 l=10 g=100 p=1 s=', '
# 32.8, 42.4, 76.8, 84.4, 23.1, 17.8, 77.3, 32.3, 60.8, 82.5
```
